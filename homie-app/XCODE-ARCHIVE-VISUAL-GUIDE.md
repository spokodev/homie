# 🎯 Як створити Archive в Xcode - Візуальний гайд

## ДЕ ЗНАЙТИ ARCHIVE В XCODE:

## 📍 СПОСІБ 1: Через меню Product (найпростіший)

### Крок 1: Виберіть пристрій
Зверху вікна Xcode, поруч з кнопкою ▶️ (Play):
```
HomieLife > [ТУТ ВИБЕРІТЬ ПРИСТРІЙ]
```
Клікніть на назву пристрою і виберіть:
- **Any iOS Device (arm64)** ← це потрібно для архіву!
- НЕ iPhone Simulator
- НЕ конкретний симулятор

### Крок 2: Створіть Archive
**В головному меню зверху екрану (не в Xcode вікні, а в macOS меню):**
1. Клікніть **Product** (між Source Control і Debug)
2. В випадаючому меню виберіть **Archive** (майже в кінці списку)

**АБО використайте комбінацію клавіш:**
- **Cmd + Shift + B**

## 📍 СПОСІБ 2: Через кнопку Play (альтернатива)

1. **Утримуйте** кнопку ▶️ (Play) в лівому верхньому куті
2. З'явиться меню:
   - Run
   - Test
   - Profile
   - Analyze
   - **Archive** ← виберіть це!

## 📍 СПОСІБ 3: Через схему (для експертів)

1. Клікніть на **HomieLife** схему (зверху зліва від кнопки Play)
2. Виберіть **Edit Scheme...**
3. Зліва виберіть **Archive**
4. Налаштуйте і закрийте
5. Потім Product → Archive

## ⚠️ ВАЖЛИВО ПЕРЕД АРХІВОМ:

### ✅ Перевірте що вибрано:
```
HomieLife > Any iOS Device (arm64)
         ↑
    НЕ симулятор!
```

### ✅ Якщо "Archive" сірий (недоступний):
1. Змініть destination на **Any iOS Device**
2. Переконайтесь що немає помилок компіляції
3. Product → Clean Build Folder (Cmd+Shift+K)

## 🎬 ЩО ВІДБУДЕТЬСЯ ПІСЛЯ:

1. **Почнеться білд** (5-10 хвилин)
2. Ви побачите прогрес зверху: "Building HomieLife | Archive"
3. **Відкриється Organizer** автоматично
4. Там буде ваш архів з датою і часом

## 💡 Швидкі кроки:

1. **Виберіть:** Any iOS Device (arm64) замість симулятора
2. **Натисніть:** Cmd + Shift + B
3. **Чекайте:** 5-10 хвилин
4. **Результат:** Organizer відкриється з архівом

## ❓ Якщо не працює:

### Archive недоступний (сірий):
- Змініть destination на "Any iOS Device"
- НЕ на симулятор!

### Помилки білду:
- Product → Clean Build Folder
- Перевірте Signing & Capabilities

### Не відкривається Organizer:
- Window → Organizer
- Або Cmd + Shift + Option + O
